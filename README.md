# Holopad

Holopad is an early-stage spatial computing prototype exploring
mathematical models, real-time graphics, and interaction design.

## Status
Prototype v0.1 — under active development

## Tech Stack
- Three.js
- React + Vite
- Netlify (hosting)

## Vision
Holopad aims to evolve from a simulation-based interface into
wearable and holographic interaction systems.
